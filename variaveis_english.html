What is varying?



Variable. What is it?

Ah… the variety! Doesn't that name seem strange to you? That bom! But if you never heard this word, or it's because he didn't attend math classes or because he wasn't on planet earth during the classes. Tricks aside, the varieties are exactly what they mean in Portuguese: variam! As they vary, therefore, their values ​​vary. One moment we can represent something, and later, we represent something else. Back to math, the most famous variable is X. Let's put a little in our memory an equation well simple ones: x + 5 = 7 And there, I ask you: what is the value of X? Well, solving the equation, I'm afraid... x + 5 = 7 x = 7 - 5 (the lembre gives the rule, or that sum of one side, passes to the other underdrawing...) x = 2 We fear that in this equation the value of x is equal to 2. Now on to another equation (and we promise to be the last one for now): 2x + 10 = x − 3 Yes, do not despair, the solution is simpler than you imagine: 2x + 10 = x - 3 (1st let's separate the x's on one side, and the isolated numbers on the other, ok?) 2x − x = −10 − 3 (now we will solve) x = −13 Here, we fear now that x is worth -13. Well, how? Before x was worth 2 and now worth -13?? Sim! Finally, x is a variable. Or yes, x has no value fixed and defined. At each moment it has a different value. Following the same idea, the variables in a computer program serve to store varied values. Let’s look at an example of a program with a variable:

program {
function begin (){
int x
x = 2
x = 3
}

Other variable types...
In addition to integers, we can create variables for other types of information:

There's four main variable types


Whole


Only those that will store hardly integer values ​​such as: 0, 1, 2, 3, 10, -1, -2, -3 Real


The only ones that will store barely real values ​​are: 0.1, -0.9, 3.14… (decimals and negatives) Characterer


Only those that will keep only letters and symbols (characters), such as... "André Roque" or "Constellation" Logic


São those that reserve barely logical values, this is, true or false, for example:

Here you may wonder, "But will it always be like this?" "Can they change?" "Can Fazer_licao be false?"... And the answer is yes! Remember? They are variables, they are made to vary according to the progress of the code. But relax...we'll get there.

Concluding
In a nutshell, variables are little boxes that help the computer remember things. Each type of variable has a different format: some store numbers, others store words, and still others can store several things at the same time. So, you can use variables to:
Store someone's age
Store a score in a game.
Store a name, like "Lucas" or "Maria".
Determine whether something is true or false, like whether or not you finished your homework.
