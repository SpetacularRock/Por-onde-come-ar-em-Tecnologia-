Estruturas Condicionais


o primeiro assunto legal

Vamos explorar estes termos...
As estruturas condicionais permitem que um programa execute diferentes comandos de acordo com as condições estabelecidas. É o famoso "se isso, então faça isso... senão faça aquilo." Elas estão presentes em diversas linguagens de programação e todo profissional da área precisa saber como utilizá-las. Elas ajudam a decidir qual caminho o programa deve seguir com base no que é verdadeiro ou falso. Isso é essencial em quase toda aplicação, já que muitas vezes precisamos fazer decisões como "Se a idade for maior que 18, então a pessoa é maior de idade", ou "Se o número é positivo, então exiba 'Positivo'". O uso dessas condições torna o código mais flexível e inteligente, pois ele pode se adaptar a diferentes situações.
Se (if)



Vamos começar do básico. A estrutura condicional mais simples é o if (se). Ele verifica uma condição e, se essa condição for verdadeira, executa um bloco de código. Se for falsa, o programa segue para a próxima linha. Às vezes, usamos o else (senão) para executar um bloco de código alternativo, caso a condição não seja verdadeira. Além disso, podemos usar o else if para verificar várias condições diferentes, oferecendo múltiplas possibilidades. Vamos fazer um exemplo para testar se uma pessoa pode tomar bebidas alcólicas baseando em sua idade: inteiro idade;
escreva ("Digite sua idade");
leia (idade);
se(idade => 18){
escreva ("Pode tomar.")
}


Neste exemplo, a variável idade é usada para armazenar a idade de uma pessoa. Se a idade for 18 ou mais, a condição if será verdadeira, e a mensagem "Pode beber" será exibida.

Else (senão)
Podemos juntar este exemplo com outra estrutura condicional, o senão. Aqui ele verificará o contrário. Se não foi atingido as espectativas:
inteiro idade;
escreva ("Digite sua idade");
leia (idade);
se(idade => 18){
escreva ("Pode tomar.")
}
senão{
escreva ("Não pode tomar.")
}

Neste exemplo simples com apenas uma variável é possível demonstrar "um zilhão" de possiblidades com as estruturas condicionais if (se) e else (senão).


Concluindo...
As estruturas condicionais permitem que o código tome decisões baseadas em condições específicas, tornando a programação mais dinâmica. Com elas é possível criar interações muito mais interessantes e complexas. O uso de if, else if e else oferece flexibilidade para implementar diferentes comportamentos dependendo do contexto, o que é essencial para criar aplicativos interativos e inteligentes.
