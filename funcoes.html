Funções


O que são?
Em programação, uma função é basicamente um bloco de código que executa uma operação e pode retornar um valor. As funções podem ter parâmetros de entrada, que permitem que os chamados passem argumentos para a função. As funções são uma das ferramentas mais poderosas e reutilizáveis em programação. Elas permitem agrupar um conjunto de instruções que realizam uma tarefa específica, dando um nome a essa sequência de ações. O grande benefício de usar funções é a possibilidade de chamar esse "bloco de código" sempre que precisarmos, sem ter que escrever o mesmo código várias vezes. Assim, funções ajudam a manter o código mais organizado, claro e reutilizável.
Uma função geralmente recebe entradas (conhecidas como parâmetros ou argumentos), executa um conjunto de instruções e, em seguida, pode retornar um valor como resultado. Essa estrutura permite que possamos passar diferentes entradas para a função e obter diferentes resultados sem mudar o código da função em si. Por exemplo, uma função que calcula a soma de dois números pode ser chamada várias vezes, passando números diferentes, e obterá um resultado diferente a cada vez.
Para declarar uma função, precisamos definir o nome dela, os parâmetros (se houver) e o que ela deve fazer. Uma vez que a função é definida, podemos chamá-la em qualquer lugar do programa para executar a tarefa que ela foi programada para realizar. Isso torna o código mais limpo, já que podemos reutilizar as funções sempre que necessário, em vez de duplicar o código

Um exemplo de função em algortimo
função Somar(a, b) {
retorne a + b;
}
Aqui essa função recebe dois parâmetros, a e b, que são os números a serem somados. Dentro da função, somamos esses dois valores e retornamos o resultado. Essa função é um molde para se utilizar em qualquer caso... O complemento deste código seria:
resultado = somar(3, 4); Neste caso, o valor 3 será passado para o parâmetro a e o valor 4 será passado para o parâmetro b. A função somar então usa esses dois valores para calcular e retornar a soma de 3 e 4. Os parênteses são os parâmetros e servem para relacionar as entradas. Na função está especificando que as entradas são a e b. E quando é chamada a função (segundo código) os parênteses são as entradas da função contextulizadas para aquela situação. Entendeu?
