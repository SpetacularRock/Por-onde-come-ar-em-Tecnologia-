Comandos de entrada e saída. O que são?
Comandos de entrada (leia) e saída (escreva) são comandos que permitem a interação entre o usuário e um programa de computador. Os comandos de entrada e saída são fundamentais para interagir com o usuário em programas de computador.
Comandos de saída
A saída de dados permite que a informação que está dentro do programa seja apresentada para o usuário, ou seja, que possamos mostrar algo que está interno. Para fazer isso no Portugol Studio, temos que utilizar o comando escreva. Vamos ver o exemplo mais clássico da programação, o famoso Hello World, ou aqui no Brasil, o Olá Mundo:
programa{
funcao inicio() {
escreva ("olá mundo!")
}
}
Contextualizei o comando dentro de um codigo fonte "inicializado", digamos. Por isso as demais linhas como "programa" e "funcao". Na linguagem Python, por exemplo, o comando seria:
print("olá mundo!");
Na linguagem C++ seria:
Na linguagem Java, seria:
System.out.println("olá mundo!");
Na linguagem Csharp seria:
Console.writeLine("olá mundo!");

Viu como que em cada linguagem escreve-se de formas variadas, porém todas seguem uma mesma "ideia", a lógica de programação, os algoritmos. Este é o comando de saída de dados. Agora anote e escreva!

Comando de entrada
Mas... o que é uma entrada de dados? Bom, todo programa de computador possui uma entrada de dados, embora nem sempre você perceba. Certamente, você já preencheu algum formulário com seu nome, selecionou a cidade em que reside, ou clicou em algum botão em seu computador. Pois bem, quando o computador pede cada uma dessas informações, foi a forma de entrarmos com dados para dentro do computador. Já o comando de entrada é usado para receber informações do usuário, ou seja, ele permite que o programa "leia" ou capture dados fornecidos por quem está utilizando a aplicação. Normalmente, usamos o comando de entrada quando queremos que o usuário digite algo, como seu nome, idade, ou até mesmo uma resposta a uma pergunta feita pelo programa. Mas... o que é uma entrada de dados? Bom, todo programa de computador possui uma entrada de dados, embora nem sempre você perceba. Certamente, você já preencheu algum formulário com seu nome, selecionou a cidade em que reside, ou clicou em algum botão em seu computador. Pois bem, quando o computador pede cada uma dessas informações, foi a forma de entrarmos com dados para dentro do computador. Esse comando geralmente se chama input na maioria das linguagens modernas (como Python), ou leia em pseudocódigos e algumas linguagens educacionais. O que o comando de entrada faz é permitir que o usuário digite algo no teclado, e o programa então usa essa informação no decorrer da execução.
Suponha que você precise pedir o nome de uma pessoa, você deve criar um programa assim:
programa{
funcao(){
caractere nome

escreva ("Digite seu nome: ")
leia (nome)
}
}
Resumindo... A saída ajuda a mostrar informações importantes, como resultados de cálculos, status do programa ou mensagens de erro, enquanto a entrada permite que o programa personalize sua execução com base nas escolhas e dados fornecidos pelo usuário. Combinando esses dois comandos de forma eficaz, conseguimos criar programas dinâmicos e interativos.
