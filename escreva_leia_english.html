Input and output commands. What are they?
Input (read) and output (write) commands are commands that allow interaction between the user and a computer program. Input and output commands are essential for interacting with the user in computer programs.

Output Commands

The data output allows the information that is inside the program to be presented to the user, that is, we can show something that is internal. To do this in Portugol Studio, we have to use the write command. Let's look at the most classic programming example, the famous Hello World!
program {
function start() {
write ("hello world!")
}
}
I contextualized the command within an "initialized" source code, let's say. That's why the other lines are "program" and "function". In Python, for example, the command would be:
print("hello world!");
In C++, it would be:
In Java, it would be:
System.out.println("hello world!");
In Csharp, it would be:
Console.writeLine("hello world!");

You see how in each language it is written in different ways, but they all follow the same "idea", the programming logic, the algorithms. This is the data output command. Now write it down and write it down!


Input Commands

But... what is a data entry? Well, every computer program has a data entry, although you may not always notice it. You have certainly filled out a form with your name, selected the city where you live, or clicked on a button on your computer. Well, when the computer asks for each of these pieces of information, it was the way we entered data into the computer. The input command is used to receive information from the user, that is, it allows the program to "read" or capture data provided by whoever is using the application. Normally, we use the input command when we want the user to type something, such as their name, age, or even an answer to a question asked by the program. But... what is a data entry? You have certainly filled out a form with your name, selected the city where you live, or clicked on a button on your computer. This command is usually called input in most modern languages ​​(like Python), or read in pseudocodes and some educational languages. What the input command does is allow the user to type something on the keyboard, and the program then uses that information during execution.
Suppose you need to ask for a person's name, you should create a program like this:
program{
function(){
character name

write ("Enter your name: ")
read (name)
}
}


To sum it up...

Output helps display important information such as calculation results, program status, or error messages, while input allows the program to customize its execution based on the choices and input provided by the user. By combining these two commands effectively, we can create dynamic and interactive programs.
