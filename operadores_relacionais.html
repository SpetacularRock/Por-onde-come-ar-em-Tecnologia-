Operadores Relacionais


relacionando o que?

O que são?
Os operadores relacionais são usados para comparar dois valores e determinar o relacionamento entre eles. Eles retornam um valor verdadeiro ou falso, dependendo de como os valores se comparam. Esses operadores são essenciais em programação para fazer decisões lógicas, como verificar se dois números são iguais ou se um é maior que o outro. Os principais operadores relacionais Igual a (==): Verifica se dois valores são iguais.
Maior que (>): Verifica se o valor da esquerda é maior que o da direita.
Menor que (<): Verifica se o valor da esquerda é menor que o da direita.
Maior ou igual a (>=): Verifica se o valor da esquerda é maior ou igual ao da direita.
Menor ou igual a (<=): Verifica se o valor da esquerda é menor ou igual ao da direita.


Tipos de operadores relacionais



Imagine que você tenha duas variáveis: A = 10 e B = 5. Vamos ver como os operadores relacionais funcionam para essas variáveis:
A == B: O operador igual a verifica se A é igual a B. Como 10 não é igual a 5, o resultado seria falso.
A != B: O operador diferente de verifica se A é diferente de B. Nesse caso, 10 é diferente de 5, então o resultado seria verdadeiro.
A > B: O operador maior que verifica se A é maior que B. Como 10 é maior que 5, o resultado seria verdadeiro.
A < B: O operador menor que verifica se A é menor que B. Como 10 não é menor que 5, o resultado seria falso. A>= B: O operador maior ou igual a verifica se A é maior ou igual a B. Como 10 é maior que 5, o resultado seria verdadeiro.
Fácil, não? Apenas uma questão de relacionar as variáveis.

Agora vamos para um exemplo simples
Esses operadores são usados principalmente em estruturas de controle como "se", para tomar decisões. Por exemplo, você pode verificar se a idade de uma pessoa é maior ou igual a 18 para saber se ela pode votar:

inteiro idade;
escreva("Qual é a sua idade?");
leia(idade);
se (idade >= 18);
escreva("Você pode votar.");
senão {
escreva("Você não pode votar.")
}
