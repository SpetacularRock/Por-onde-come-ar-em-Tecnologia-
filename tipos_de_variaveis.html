Outros tipos de variáveis...
Então como vimos anteriormente, (vá ao final da página e clique na seta localizada no canto inferior esquerdo caso queira reler) em programação uma variável é como uma "caixa" onde podemos armazenar informações temporárias enquanto o programa está rodando. Pense nela como uma etiqueta que você coloca em um objeto para identificar o que ele contém. Essa "caixa" pode armazenar diferentes tipos de informações, como números, textos, ou até mesmo valores lógicos (verdadeiro ou falso). Para saber qual tipo de informação uma variável pode armazenar, utilizamos os tipos de variáveis. Esses tipos definem qual tipo de dado pode ser guardado naquela "caixa" e qual tipo de operação podemos realizar sobre ele. Além de números inteiros, podemos criar variáveis para outros tipos de informações:
Existem 4 tipos principais de variáveis:
Inteiras
São aquelas que vão guardar apenas valores inteiros como: 0, 1, 2, 3, 10, -1, -2, -3... Reais
São aquelas que vão guardar apenas valores reais, isto é: 0.1, -0.9, 3.14... (decimais e negativos) Caractere
São aquelas que vão guardar apenas letras e simbolos (caracteres), como... "André Roque" ou "Constelação" Lógicas
São aquelas que reservam apenas valores lógicos, isto é, verdadeiros ou falsos, por exemplo:
logico fazer_licao = true;
Nesta linha declarei uma variavel logica chamada "fazer_licao" e atribui um valor verdadeiro a ela.

Aqui pode-se surgir a dúvida de "mas será sempre assim?" "Elas podem mudar?" "Fazer_licao pode ser falso?"... E a resposta é sim! Lembra? São variáveis, são feitas para variar conforme o andamento do código. Mas relaxe...vamos chegar lá.

Inteiras
Inteiro (int): O tipo inteiro é usado para armazenar números inteiros, ou seja, números sem casas decimais. Por exemplo, o número 5, o número -3 ou o número 200 são números inteiros. Variáveis desse tipo são úteis quando você precisa trabalhar com contagens, idades, pontuações, ou qualquer situação em que não faça sentido usar números decimais.
Reais
O tipo real é utilizado para armazenar números com casas decimais ou negativos, ou seja, números que não são inteiros. Esse tipo é muito usado quando precisamos de mais precisão em cálculos, como no caso de medidas, preços e coordenadas geográficas. Por exemplo, 3.14, -0.5, 100.75 são números de ponto flutuante.
Caractere
O tipo caractere (texto) é usado para armazenar textos, ou seja, qualquer sequência de caracteres. Isso inclui palavras, frases e até mesmo números que você queira tratar como texto (por exemplo, um número de telefone ou um código postal). As strings são sempre colocadas entre aspas (simples ou duplas).
Lógico
Já a variável lógica armazena valores lógicos, ou seja, verdadeiro (True) ou falso (False). Esse tipo é muito importante quando usamos condições no programa, onde precisamos tomar decisões baseadas em comparações. Por exemplo, uma variável booleana pode representar se uma pessoa é maior de idade ou se uma resposta está correta em um teste. Array
O array (ou lista) é usado para armazenar uma coleção de dados. Diferente dos outros tipos, uma lista pode armazenar múltiplos valores em uma única variável. Esses valores podem ser de qualquer tipo (inteiro, string, etc.), e você pode acessar cada valor da lista pelo seu índice. As listas são úteis quando você precisa lidar com conjuntos de informações, como uma lista de nomes ou uma sequência de números. Por exemplo:
numeros = [1, 2, 3, 4, 5]
