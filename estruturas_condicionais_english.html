Conditional Structures


the first cool topic

Let's explore these terms...
Conditional structures allow a program to execute different commands based on established conditions. It's the famous "if this, then do that... otherwise do this." They are present in many programming languages, and every professional in the field needs to know how to use them. They help decide which path the program should take based on what is true or false. This is essential in almost every application, since many times we need to make decisions like "If the age is greater than 18, then the person is an adult," or "If the number is positive, then display 'Positive'". Using these conditions makes the code more flexible and intelligent, as it can adapt to different situations.
If



Let's start with the basics. The simplest conditional structure is the `if`. It checks a condition, and if that condition is true, it executes a block of code. If it's false, the program moves on to the next line. Sometimes, we use `else` to execute an alternative block of code if the condition is not true. Additionally, we can use `else if` to check multiple different conditions, offering several possibilities. Let's make an example to test if a person can drink alcoholic beverages based on their age: integer age;
write ("Enter your age");
read (age);
if(age >= 18){
write ("You can drink.")
}


In this example, the variable `age` is used to store a person's age. If the age is 18 or older, the `if` condition will be true, and the message "You can drink" will be displayed.

Else
We can combine this example with another conditional structure, `else`. Here, it will check the opposite. If the expectations are not met:
integer age;
write ("Enter your age");
read (age);
if(age >= 18){
write ("You can drink.")
}
else{
write ("You cannot drink.")
}

In this simple example with just one variable, it is possible to demonstrate "a million" possibilities with the `if` and `else` conditional structures.


In conclusion...
Conditional structures allow the code to make decisions based on specific conditions, making programming more dynamic. With them, it is possible to create much more interesting and complex interactions. The use of `if`, `else if`, and `else` provides flexibility to implement different behaviors depending on the context, which is essential to create interactive and intelligent applications.
