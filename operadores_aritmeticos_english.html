Operators. What are they?
Arithmetic operators are symbols that allow you to perform mathematical operations, such as addition, subtraction, multiplication, division, etc., within the code. They are essential for any type of calculation and are used to manipulate numerical values. Arithmetic operators are those that allow you to perform the basic operations of mathematics. Come on, I know you remember them ;-)
Sum (plus): allows you to accumulate two or more values. In programming, it is represented by the symbol +. Example:
integer value1, value2
value1 = 2
value2 = 5
result = value1 + value2 Subtraction (minus): allows you to remove two or more values. In programming, it is represented by the symbol -. Example:
integer value1, value2
value1 = 2
value2 = 5
result = value1 - value2
Multiplication (times): allows you to multiply two or more values. In programming, it is represented by the symbol *. Example:
integer value1, value2
value1 = 2
value2 = 5
result = value1 * value2
Division (divide): division is understood by two operations, division and remainder. In other words, in programming, it is possible to obtain the result of these two operations, just like in a normal calculation. To perform division, we use the symbol / . If the division involves integers, it will return the integer value of the division. If it is a real number, it will return the value of the division with decimal places. To obtain the remainder, we use the symbol %. Remembering that the remainder always occurs in relation to an integer division. Example:
integer value1, value2
value1 = 2
value2 = 5
division = value2 / value1
remainder = value2 % value1
Furthermore, the priority of operators must be observed. Just like in traditional mathematics, in programming the same priorities are respected. See the examples:
Example 1: 3 + 7 * 2 => 3 + 14 => 17
Example 2: (3 + 7) * 2 => 10 * 2 -> 20
In other words, we always perform what is in parentheses first. Then, multiplications and divisions. And finally, additions and subtractions. In the case of more than one operation of the same type, the priority is from left to right.
To sum it up... Addition (+): Adds two values.
Subtraction (-): Subtracts the second value from the first.
Multiplication (*): Multiplies two values.
Division (/): Divides the first value by the second. (In some languages, such as C++, dividing
integers can result in another integer, discarding the decimal part.)
Modulus (%): Returns the remainder of the division between two numbers.
