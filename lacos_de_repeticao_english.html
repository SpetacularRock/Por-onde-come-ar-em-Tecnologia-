Loops


What are they?
Loops are very important control structures in programming. They allow a block of code to be executed multiple times based on a specific condition. Instead of writing the same code repeatedly, you can use a loop to save time and make your code more efficient. There are three main types of loops: the `for` loop, the `while` loop, and the `do-while` loop. Each one has a different way of controlling the number of repetitions, but all aim to repeat an action while a condition is true.
The `for` loop is typically used when we know exactly how many times we want to repeat the code. It consists of three parts: initialization, the repetition condition, and increment or decrement. These parts are checked at each iteration, and the loop continues executing until the defined condition becomes false. The `while` loop is ideal when we don’t know the exact number of repetitions. It repeats the code block while the condition is true, and the condition is checked before each execution. The `do-while` loop is similar to the `while` loop, but the condition is checked after the code execution, ensuring that the code block is executed at least once.
Example:

Suppose we want to print the numbers from 1 to 5 on the screen. Using the `for` loop, we can do this in a concise and clear way:

for (i from 1 to 5) do{
write(i)}


In this example, the loop starts with i = 1 and goes up to i = 5, because the range(1, 6) function generates the numbers from 1 to 5 (the number 6 is not included). With each iteration, the value of i increases by 1 (this is the increment), and the value of i is printed on the screen. When i reaches 6, the loop condition (i < 6) becomes false, and the loop ends.
