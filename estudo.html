Algoritmo, codigo fonte e programa


qual a diferença?

Vamos explorar estes termos...
No mundo da tecnologia é comum, principalmente no início utilizar nomeclaturas como algoritmos, programas e códigos fontes... "Isto é um programa de computador!" "Dê uma olhada neste código!"
Todas estas nomeclaturas vem do estudo de lógica de programação, o primeiro assunto a ser tratado quando se fala de técnologia.

O que é um algoritmo?
Utilizamos a lógica de programação para criar algoritmos, códigos (códigos-fonte) e programas. Vamos especificar cada um destes conceitos...
Bom, é tudo quase a mesma coisa... um programa de computador é um conjunto de comandos que será realizado pelo computador. Esse conjunto de comandos deve seguir um raciocínio lógico, o que na matemática é chamado de algoritmo. Esse algoritmo é algo mais alto nível (isto é, fácil de compreender, longe da linguagem de máquina e próximo da linguagem humana) algo que você pode explicar para seus avós ou para seu irmão mais novo, que eles vão entender. Agora, para que seu computador entenda, é preciso que o algoritmo seja escrito em forma de código-fonte. E por fim, um programa é o código fonte no contexto de execução, ou seja, quando ele está funcionando, sendo executado por um computador (é um aplicativo, site...). Também pode receber o nome de Software. Ou seja, no fundo, no fundo, tudo é a mesma coisa!
Algoritmos resumem-se em ser um conjunto ordenado de passos lógicos. Algoritmos são simplesmente um conjunto de passos ou instruções que você segue para fazer algo ou resolver um problema. Vamos comparar com algo bem familiar: Fazer um bolo
Imagine que você irá fazer um bolo. Para isso há uma série de passos/processos para conseguir fazer um bolo:
1. Reserve os ingredientes e recipientes
2. Bata claras em neve e reserve
3. Misture as gemas, a margarina e o açúcar até obter uma massa homogênea
4. Acrescente o leite e a farinha de trigo aos poucos, sem parar de bater.
5. Adicione as claras em neve e o fermento
6. Despeje a massa em uma forma untada
7. Asse o bolo no forno
8. Espere 40 minutos
9. Retire o bolo

O que é código fonte?
O código fonte é a contextualização destes passos a passos em uma linguagem de programação (em um idioma, digamos.). É como se os algoritmos fossem a gramatica do programa e o códgio fonte a linguagem (as linguas partilham semelhanças de estruturas sintáticas, morfológicas, nonminais... Isto é, todas elas tem verbos e conjugações mas todas utilizam outras palavras para se expressarem).
Então, o código fonte é a lista de coisas que o programador escreve para o computador fazer. Por exemplo, ele pode dizer ao computador: "Mostre uma imagem", "Toque uma música" ou "Abra uma página na internet". Essas instruções são escritas em uma linguagem especial que o computador consegue entender, como se fosse um idioma secreto entre as pessoas que fazem os programas e o computador.


O que é um programa de computador?

Agora, pensa em um programa como se fosse o prato final que você faz seguindo a receita. O programa é o resultado de tudo o que o computador faz seguindo as instruções do código fonte. Então, quando o programador escreve o código fonte, ele está criando um programa, que é o produto final. Por exemplo, um jogo de videogame é um programa, e a página de internet onde você vê fotos de animais também é um programa. Então resumindo, algoritmos são os passos a passos de forma que você entende (em português), codigos são estes passos a passos de maneira que o computador entenda e programas são os codigos sendo executados, o produto final.
